# Calendar

Simple calendar service

## Prerequisites

```bash
Go >=1.13.6
```

### Installing

```bash
git clone https://github.com/tima-fey/otus-golang/tree/master/calendar
go build

```

## Run

```bash

./calendar -config config.json
```
